# Pattern Trace

Deep pattern archaeology. Pure Awareness stage work:

1. **Name the pattern or behavior you want to trace.** Be specific.
2. **When does this show up?** With whom? In what situations?
3. **Now go back:** When was the first time you remember this pattern showing up? What age were you?
4. **Which parent did you learn this from?** (Read `references/pattern-catalog.md` for the full taxonomy to help identify)
5. **Was it Adoption, Reaction, or Rebellion?**
   - *Adoption*: you do what they did
   - *Reaction*: you repress or protect against what they did
   - *Rebellion*: you try to be the opposite
6. **What was the ORIGINAL NEED this pattern was trying to meet?** Every pattern started as a survival strategy.
7. **Who did your PARENT learn it from?** (Their parents — Negative Love Syndrome goes back generations)
8. **How does this pattern serve you now?** Does it still protect you, or has it become the prison?

If personal profile exists, connect to the specific parent dynamics and family history.

Read `references/pattern-catalog.md` to help identify the category and related patterns.
